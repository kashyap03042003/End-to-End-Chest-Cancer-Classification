# End-to-End Chest Cancer Classification

A production-ready deep learning project for chest cancer classification using Convolutional Neural Networks (CNN).

## Project Structure

- `src/cnnClassifier/` - Main package
- `config/` - Configuration files
- `research/` - Research notebooks
- `templates/` - Web templates

## Installation

```bash
pip install -r requirements.txt
```

## Usage

This project uses DVC for data versioning and MLFlow for experiment tracking.
